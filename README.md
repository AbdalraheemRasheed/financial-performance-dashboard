
# Financial Performance Dashboard — NovaTech Solutions

Financial analysis project focused on revenue, profit, and performance tracking
using Python, SQL, and Excel to uncover insights and support business decision-making.

## 📈 Dashboard Preview
[financial_data_clean_excel.pdf](https://github.com/user-attachments/files/26982439/financial_data_clean_excel.pdf)


## 📊 Key Insights
- Electronics leads in units sold and contributes the most to total revenue.
- South is the strongest region with the highest profit.
- East is the weakest region and needs focus for growth.
- Profit margin is stable at 34% across all categories.
- Sara is the top sales rep with $318,052 in revenue.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- MySQL
- Excel (Pivot Tables, Charts)

## 📊 Analysis Includes
- Data cleaning & preparation
- KPI calculation (revenue, profit, margin)
- Regional and category performance analysis
- SQL queries for data exploration
- Pivot tables and charts in Excel
- Matplotlib visualizations

## 🚀 How to Run
1. Clone the repository
2. Install requirements: pip install pandas matplotlib openpyxl
3. Open financial_data_clean.ipynb in VS Code to view the Python analysis
4. Import financial_data_clean_sql.sql into MySQL to explore the data
5. Open financial_data_clean.xlsx to explore the pivot tables and charts
